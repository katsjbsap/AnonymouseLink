# anonymousLink module (MVP skeleton)

This is a TypeScript skeleton implementing the proposed architecture (domain/application/infrastructure/presentation/jobs).

## Run (dev)
```bash
npm i
npm run dev
```

## Endpoints (example)
- POST /links
- GET /a/:token
- POST /a/:token/auth
